**Programming Language Cryptography (PLC)** adalah metode kriptografi sederhana yang memanfaatkan **bahasa pemrograman** sebagai basis utama pembentukan kunci enkripsi.
Metode ini menggabungkan unsur **linguistik**, **data publik**, dan **transformasi numerik** untuk menghasilkan ciphertext yang **unik, konsisten, dan mudah direproduksi**.

PLC dirancang sebagai pendekatan kriptografi yang **dekat dengan dunia programmer**, sehingga cocok digunakan sebagai media pembelajaran kriptografi dasar.

Metode PLC menggunakan beberapa komponen utama:

   * Nama penemu bahasa
